
## JPA (Java Persistence API)

JPA는 자바에서 제공하는 ORM (Object-Relational Mapping) 기술에 대한 표준 API를 지칭한다. ORM 기술은 객체 지향 프로그래밍 언어와 관계형 데이터베이스 간의 호환되지 않는 데이터를 변환하는 프로그래밍 방식을 제공. 

## ORM (Object-Relational Mapping)

ORM은 객체와 관계형 데이터베이스의 데이터를 매핑하는 기술로, 개발자가 객체 지향 언어를 사용하여 데이터베이스를 보다 직관적으로 다룰 수 있게 해준다. 예를 들어, 자바와 같은 객체 지향 언어를 사용하고 데이터베이스에 주로 사용하는 SQL의 경우, 이 두 언어 간의 직접적인 연동이 어렵기 때문에 ORM이 필요하다.

ORM을 사용하지 않을 경우, 자바 개발자는 매번 자바 객체와 SQL 사이의 변환을 직접 구현해야 한다.

## 하이버네이트 (Hibernate)

하이버네이트는 JPA의 명세를 구현한 대표적인 자바 ORM 라이브러리 중 하나입니다. JPA 자체는 구현 기술이 아니라 ORM을 위한 API 명세를 제공하는 표준 규격. 따라서, JPA를 사용함으로써 다양한 ORM 기술을 일관된 방식으로 접근할 수 있습니다.
